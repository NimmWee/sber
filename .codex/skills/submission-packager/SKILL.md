# Goal
Prepare the final submission as a stable, explainable, reproducible package.

When invoked:
1. Assemble final score(prompt, response) pipeline.
2. Ensure:
   - smoke tests pass,
   - benchmark script runs,
   - artifact paths are stable,
   - README explains architecture and trade-offs.
3. Produce a final checklist before submission.
